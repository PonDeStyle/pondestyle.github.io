ダウンロード & 導入

PonDeStyleの導入方法について、説明します。

ダウンロード

1. CDN

CDNから読み込む方法を現在準備中です。

<link rel="stylesheet" href="###">

2. npm

npmレポジトリは現在準備中です。

npm install pondestyle

または、

yarn add pondestyle

3. GitHubからダウンロード

GitHubレポジトリからダウンロードできます。

PostCSSでビルド

PostCSSのソースコードをビルドすることで、容易なカスタマイズが可能になります。
