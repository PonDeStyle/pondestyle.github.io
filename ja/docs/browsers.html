サポートブラウザーと注意事項

PonDeStyleは以下のブラウザーをサポートします。

  • Internet Explorer 11
  • Firefox (Desktop, Android)
  • Chrome (Desktop, Android)
  • Edge (現在のところ、Desktop のみ）
  • Safari (Desktop, iOS)

注意事項

  • Autoprefixer で -ms- 接頭辞をつけても、IE10 では正しく表示できません。
  • IE 11 以外のブラウザーは２つ前までのバージョンをサポートします。ただし、Firefox は ESR 版もサポートします。
  • Desktop ブラウザーは Windows および macOS をサポートします。

自分でビルドする場合、サポートの範囲が異なる可能性があります。たとえば postcss-cssnext で customProperties の変換を無効にしてビルドした場合、対応ブラウザーは Edge 15 以降、Firefox、Chrome、Edge、Safari となります。

将来のバージョンで、古いブラウザーのサポートを終了する可能性があります。
