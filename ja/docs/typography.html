小見出し

<hgroup> 要素内の <h2> から <h6>、もしくは .subhead クラスで小見出しやサブタイトル、短いリード文を表現します。

This is heading.

And this is sub heading.

<hgroup>
    <h1>This is heading.</h1>
    <h3>And this is sub heading.</h3>
</hgroup>

もしくは、W3C 仕様に準拠した書き方:

This is heading.

And this is sub heading.

<header>
    <h1>This is heading.</h1>
    <p class="h3 subhead">And this is sub heading.</p>
</header>
