Languages

グローバル・日本語・韓国語・中国語のフォント設定。

Languages/
  |
  |- global.css /* グローバルフォント設定 */
  |- ja.css /* 日本語 */
  |- kr.css /* 韓国語 */
  |- zh-cn.css /* 中国語 (簡体中文) */

フォントファミリーには、以下の4つのバリエーションがあります。

  • 通常
  • セリフ体
  • 整形済み要素、コード用
  • ボタン・アラート等のコンポーネント用

例

通常

Lorem ipsum dolor sit amet, fringilla mauris mauris viverra viverra. Sed quia laoreet suspendisse ut libero, euismod tristique adipiscing, et sit habitasse vitae ipsum ultrices.

セリフ体

Lorem ipsum dolor sit amet, fringilla mauris mauris viverra viverra. Sed quia laoreet suspendisse ut libero, euismod tristique adipiscing, et sit habitasse vitae ipsum ultrices.

セリフ体を使用するには、.serif クラスを付与します。

整形済み要素・コード

blockquote {
    border-left: .4rem solid #464f55;
    font-weight: var(--font-weight-medium);
    padding-bottom: .4rem;
    padding-left: 1rem;
}

pre 要素、code 要素、kbd 要素、var 要素、そして samp 要素に適用されます。

コンポーネント用

This is alert.

Lorem ipsum dolor sit amet, fringilla mauris mauris viverra viverra. Sed quia laoreet suspendisse ut libero, euismod tristique adipiscing, et sit habitasse vitae ipsum ultrices.

macOS 環境で -apple-system が、Windows の日本語環境で Yu Gothic UI が使われる点が通常のフォント設定と異なります。

ビルトインのコンポーネント以外でこのフォント設定を利用するには .uifont クラスを使用します。
