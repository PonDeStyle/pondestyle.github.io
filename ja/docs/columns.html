Columns

カラムシステムについて説明します。

概要

PonDeStyleには、他のフレームワークによくある12列などの固定グリッドが存在しません。CSS Flexible Box Layout を用いた簡易的で柔軟なレイアウトシステムを採用しています。applyルールを用いることで、独自のクラス名でさらに詳細なレイアウト構築が可能になります。

基本的な使い方

マルチカラムレイアウトを構築するには、.columns もしくは .C クラスを持つFlexコンテナと、.columns__item もしくは .c クラスを持つFlexアイテムを使用します。

1
2
3
<div class="C">
    <div class="c"></div>
    <div class="c"></div>
    <div class="c"></div>
</div>

あるFlexアイテムに、他のアイテムの2倍の幅を持たせたい場合は、 .columns__item--2x クラスを追加します。

1
2
3
<div class="C">
    <div class="c"></div>
    <div class="c c--2x"></div>
    <div class="c"></div>
</div>

ハイフンが2個ついた modifier を持つクラスは、それを持たないクラスの後に続けて記述します。これは、 .c クラスの 2x オプションが true であることを示します。

通常、タブレット未満の画面幅（768px未満）ではカラムは折り返します。タブレット未満でも横並びを維持するには、Flexコンテナに .columns--mobile クラスを追加します。

アイテムを折り返すには、Flexコンテナに .columns--wrapped クラスを追加します。

応用的な使い方

Apply ルールを用いて独自のマルチカラムクラスを定義する方法を説明します。

まず最初に、Flexコンテナに --columns-initial を、Flexアイテムに --columns-item-initial を適用します。

.mycolumn {
    @apply --columns-initial;

    &__item-1 {
        @apply --columns-item-initial;
    }

    &__item-2 {
        @apply --columns-item-initial;
    }
}

次に、それぞれの画面幅でのアイテムの幅を指定します。たとえば上記の例で、タブレット以上デスクトップ未満の場合に 1:2、デスクトップ以上の場合に 2:3 の比率でアイテムを表示したい場合は、以下のようにapplyルールを適用します。

.mycolumn {
    @apply --columns-initial;

    &__item-1 {
        @apply --columns-item-initial;
        @apply --columns-item-1x-tablet;
        @apply --columns-item-2x-desktop;
    }

    &__item-2 {
        @apply --columns-item-initial;
        @apply --columns-item-2x-tablet;
        @apply --columns-item-3x-desktop;
    }
}

モバイル幅でも横並びを維持する場合は、以下のようにします。

.mycolumn {
    @apply --columns-initial;
    @apply --columns-keep-mobile;

    &__item-1 {
        @apply --columns-item-initial;
        @apply --columns-item-3x-keep-mobile;
        @apply --columns-item-2x-tablet;
        @apply --columns-item-1x-desktop;
    }

    &__item-2 {
        @apply --columns-item-initial;
        @apply --columns-item-1x-keep-mobile;
        @apply --columns-item-3x-tablet;
        @apply --columns-item-1x-desktop;
    }
}

以下は、複数のアイテムを1行につき、モバイル幅で1つ、タブレット幅で2つ、デスクトップ以上で3つ表示する例です。

.anothercolumn {
    @apply --columns-initial;
    @apply --columns-wrapped;

    &__item {
        @apply --columns-item-initial;
        @apply --columns-item-half-tablet;
        @apply --columns-item-third-desktop;
    }
}

Applyルールリファレンス

Flexコンテナ

Applyルール 説明
--columns-initial Flexコンテナを定義します。
--columns-keep-mobile モバイル幅でも折り返さずカラムを維持します。
--columns-column カラムを上下に配置します。
--columns-wrapped はみ出したアイテムを次の行に折り返します。

Flexアイテム

Applyルール 説明
--columns-item-initial Flexアイテムを定義します。
--columns-item-'アイテムの幅'-'画面幅' それぞれの画面幅での、アイテムの幅を指定します。

アイテムの幅の一覧

アイテムの幅 説明
1x 他のアイテムの1倍の幅を持とうとします。
2x 他のアイテムの2倍の幅を持とうとします。
3x 他のアイテムの3倍の幅を持とうとします。
4x 他のアイテムの4倍の幅を持とうとします。
half コンテナの半分の幅を持とうとします。
third コンテナの3分の1の幅を持とうとします。
quarter コンテナの4分の1の幅を持とうとします。

画面幅の一覧

画面幅 説明
keep-mobile 768px 未満。ただし、Flexコンテナに keep-mobile が設定されている必要があります。
tablet 768px 以上 992px 未満
desktop 992px 以上 1200px 未満
wide 1200px 以上
