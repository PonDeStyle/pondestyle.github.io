Support browsers

PonDeStyle is supported following browsers

  • Internet Explorer 11
  • Firefox (Desktop, Android)
  • Chrome (Desktop, Android)
  • Edge (For now, only Desktop)
  • Safari (Desktop, iOS)

Note

  • IE 10 is not supported even if you use autoprefixer -ms- prefix.
  • Latest 2 versions support.
  • We are tested on macOS and Windows.

In future versions, we may end support for older browsers.
